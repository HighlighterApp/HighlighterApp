# Highlighter Apps

Modern React + Tailwind CSS website for Highlighter apps - instant offline montage making apps.

## Apps

- **Highlighter** - Core experience for recording sessions, marking highlights, and exporting montage videos
- **Highlighter:edit** - Focused editing tools for trimming and polishing highlight clips

## Features

- 🎨 Wired magazine-inspired design
- 🌓 Dark/Light mode toggle
- 📱 Fully responsive
- ⚡ Built with React + Vite + Tailwind CSS
- 🔗 Icons with Lucide React

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
